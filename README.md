<h1>Go: Jogo de Tabuleiro</h1>

<p>
  Go é um jogo de tabuleiro de estratégia abstrata para dois jogadores onde o objetivo é cercar mais território no tabuleiro do que seu oponente.
  O jogo é jogado em uma grade de interseções, e os jogadores se revezam colocando pedras pretas ou brancas nessas interseções.
  O primeiro jogador a mover é o preto. O jogo termina quando ambos os jogadores passam a vez, e o vencedor é o jogador com mais território.
</p>
